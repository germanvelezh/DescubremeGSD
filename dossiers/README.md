# dossiers/

Research consolidado de cada instrumento psicometrico del stack v1.5. Un dossier por test, con evidencia psicometrica completa.

**Tier:** 2 (cargar solo cuando se trabaje el test).

---

## Que contiene un dossier

Cada `NN_<CODIGO>_Consolidado.md` incluye:

- Constructo medido y modelo teorico
- Numero de items, dimensiones, facetas
- Escala Likert / tipo de respuesta
- Tiempo de administracion
- Propiedades psicometricas: α de Cronbach, ω, CFA (CFI, RMSEA, SRMR), invariancia
- Validacion en LATAM / es-CO si existe
- Limitaciones y sesgos conocidos
- Items literales (si la licencia lo permite)
- Claves inversas
- Scoring por dominio / faceta
- Baremos disponibles por pais / sexo / edad
- Texto de interpretacion por banda
- Adaptaciones culturales (es-CO, es-MX, en, etc.)
- Licencia y permisos
- Referencias APA 7

## Convencion de nombres

`NN_<CODIGO>_Consolidado.md` donde:
- `NN` = numero de orden en el stack (01-25+).
- `<CODIGO>` = sigla del instrumento (BFI-2, VIA-IS-P, etc.).

Si una version distinta del mismo test (ej. BFI-2 60 vs BFI-2-S 30) requiere dossier propio, sufijar: `01_BFI-2-S_Consolidado.md`.
